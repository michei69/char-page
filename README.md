# char page

a simple react webapp inspired by characterhub that i wrote like 2 years ago

it's insecure as fuck since its meant for local / private usage

## dev

### Requirements

- bun or node
- bun or (p)npm or yarn or wtv
- any browser
- s3

### setup

1. clone the repository
2. `bun i`
3. `bun dev`
4. open in a browser that doesnt yell about cors

### prod

1. `bun run build`
2. `bun serve`
